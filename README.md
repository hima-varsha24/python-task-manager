# 📝 Python Task Manager

A simple command-line based Task Manager built using Python.  
This project allows users to add, view, complete, and delete tasks efficiently.

---

## 🚀 Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Simple and beginner-friendly code

## 🛠️ Technologies Used
